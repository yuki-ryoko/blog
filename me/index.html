　　最近因为再次日常重播，所以晚饭后还是躺在床上看了大半个月。
　　之前没有看过原配，直接上的国语，而且千寻配得挺不错，加上人设，算是中意的那种类型。
　　一路下来，最后一集桂马家门前的这段 BGM 给我的印象非常深刻。我决定挂首页一段时间 (虽然没人听)。
　　欢迎来到 Silica Archives。
　　这是一个挂载在 GitHub Pages 的 Jekyll 站点，作为 Kazumi Blog 的归档点，包含原站点的部分记录。
　　我一直很喜欢这种复古的 Windows 风，每次见到都似乎回到儿时那种对于 PC 世界无限向往的时光，每天启动 Windows 98 都充满了乐趣与期待。
　　当然，时光一去不复返，所以当老友记把这 Jekyll 主题发给我的时候我就决定来一个归档点了，鸡蛤足够安全，不怕文档丢失。

　　西莉卡超可爱！所以备用站点取了 Silica 的名字。
　　「看着努力学习的你，就能让我大大地鼓起勇气。本来只是冷漠地看待的进路现在变得这么具体，至今为止总是觉得『很困难吧』的司法考试也是这样，我的目标都逐渐改变了。」
　　「都是拜你所赐喔」，小爱笑道。
